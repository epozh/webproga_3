## Функционал

### Игровая механика
- Сетка 4x4 с динамической генерацией
- Слияние плиток одного номинала
- Появление новых плиток (2 или 4) после каждого хода
- Подсчёт очков (сумма объединённых плиток)
- Проверка окончания игры (нет доступных ходов)

### Управление
- **Desktop**: стрелки на клавиатуре
- **Mobile**: свайпы по игровому полю или виртуальные кнопки

### Дополнительные функции
- Отмена последнего хода (Undo)
- Сохранение состояния игры в localStorage
- Таблица лидеров (топ-10) с сохранением в localStorage
- Адаптивный дизайн для всех устройств
- Плавная анимация движения и появления плиток

## Технологии
- HTML5 (семантическая разметка)
- CSS3 (Flexbox, анимации, медиазапросы)
- JavaScript (ES5/ES6, DOM API, localStorage)
